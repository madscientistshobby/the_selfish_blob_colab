# the_selfish_blob_colab

Colab-only companion for **[the_selfish_blob](https://github.com/madscientistshobby/the_selfish_blob)** (Mac / desktop Pygame version).

Open in Colab: [Colab.ipynb](https://colab.research.google.com/github/madscientistshobby/the_selfish_blob_colab/blob/main/Colab.ipynb)

The notebook clones `the_selfish_blob`, copies this repo's `colab_run.py` into it, and runs headless (matplotlib plots + optional GIF).
